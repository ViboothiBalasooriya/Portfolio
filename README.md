# 🚀 Modern Multi-Role Portfolio

A premium, responsive portfolio featuring dual-role experiences: **Full Stack Developer** and **UI/UX Designer**. This project showcases a clean landing page with a split-screen interaction, leading to two distinct, specialized portfolio versions.

---

## 🌟 Live Demo
🔗 [vibothibalasooriya.me](https://vibothibalasooriya.me)

---

## ✨ Features

### 🏛️ Landing Page
- **Split-Screen Layout**: Interactive hover effects that emphasize the choice between 'Developer' and 'Designer'.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.
- **Glassmorphism UI**: Modern aesthetic with blurred backgrounds and sleek borders.

### 💻 Developer Portfolio
- **Project Showcase**: Detailed cards for software engineering projects.
- **GSAP Animations**: Smooth transitions and scroll-triggered animations.
- **Micro-interactions**: Hover states and interactive elements for better UX.

### 🎨 Designer Portfolio
- **Visual-First Layout**: Emphasis on high-quality design assets and case studies.
- **Unique Typography**: Custom fonts (Outfit & Syne) tailored for creative expression.
- **Intuitive Navigation**: Single-page layout with smooth scrolling.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Advanced styling with Flexbox, Grid, and Animations.
- **JavaScript**: Core logic and interactivity.
- **GSAP**: Industry-leading animation library for high-performance motion.
- **Google Fonts**: Premium typography (Inter, Sora, Outfit, Syne).
